How Team Mode works
  • Host creates a room and shares the 4-letter code.
  • Tournament can start once 3+ players are connected.
  • Players join on their phones or tablets using the code.
  • Host's screen shows the cards. Players use their devices to call correct/incorrect.
  • Works on the same Wi-Fi or same browser (multiple tabs).
  • No accounts. No tracking. Everything stays on-device.

Beta 2 · Nearby sync uses BroadcastChannel API (same browser, same device, or same browser extension). Full cross-device support coming in a future update.
